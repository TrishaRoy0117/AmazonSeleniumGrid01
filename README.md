# Amazon Selenium Grid Project

This project performs an automated search on Amazon using Selenium Grid.

## Technologies Used
- Selenium WebDriver
- Selenium Grid (Hub & Firefox Node on Docker)
- TestNG
- Maven
- Java
- Docker Compose

## Project Structure

├── pom.xml
├── testng.xml
├── src/test/java/amazonTest

│   └── AmazonSearchTest.java
